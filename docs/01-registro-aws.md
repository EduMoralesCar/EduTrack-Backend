# Guía 01: Registro en AWS (Amazon Web Services)

Esta guía detalla el proceso paso a paso para crear una cuenta gratuita en AWS para alojar el sistema de gestión académica **EduTrack**.

---

## 1. Requisitos Previos
*   Una dirección de correo electrónico activa.
*   Un número de teléfono móvil para verificación por SMS.
*   Una tarjeta de débito o crédito (física o digital/prepago) con al menos **$1.00 USD** de saldo para verificar la identidad. *AWS cargará un dólar temporal que será reembolsado inmediatamente*.

---

## 2. Proceso de Registro Paso a Paso

1.  **Ingresar al portal de registro:**
    *   Ve a [aws.amazon.com/free](https://aws.amazon.com/free) y haz clic en **"Crear una cuenta de AWS"** (Create an AWS Account).

2.  **Información de inicio de sesión:**
    *   Ingresa tu correo electrónico y un nombre para tu cuenta de AWS.
    *   Verifica tu correo con el código numérico enviado.

3.  **Configurar contraseña:**
    *   Crea una contraseña segura y guárdala en un gestor seguro.

4.  **Información de contacto:**
    *   Selecciona el tipo de cuenta **Personal** (ideal para proyectos académicos).
    *   Llena tus datos personales (nombre, teléfono, dirección y código postal).

5.  **Información de Facturación:**
    *   Ingresa el número de tu tarjeta de crédito o débito.
    *   *Nota: AWS no realizará ningún cobro durante tu periodo de prueba siempre que te mantengas bajo los límites de la capa gratuita.*

6.  **Verificación de Identidad por SMS:**
    *   Ingresa tu número de teléfono y el código de verificación que recibirás por mensaje de texto.

7.  **Selección del Plan de Cuenta (¡Crucial!):**
    *   AWS te ofrecerá dos planes de cuenta:
        *   **Gratuito (6 meses / Promo):** Ofrece un crédito inicial (por ejemplo, **$120 USD** o **$200 USD**) y se cierra automáticamente al finalizar el periodo sin realizar cargos imprevistos. *(Recomendado para evitar cobros sorpresa)*.
        *   **De pago (Capa gratuita estándar):** Requiere cambio manual a pago si te excedes de los recursos.
    *   Elige el **Plan Gratuito (Free)**.

---

## 3. Acceso a la Consola de Administración
Una vez completado el registro:
1.  Haz clic en **"Iniciar sesión en la consola"** (Sign in to the Console).
2.  Inicia sesión como **Root User** (Usuario raíz) con el correo y contraseña registrados.
3.  En la esquina superior derecha, selecciona la región de despliegue preferida (por ejemplo, **N. Virginia `us-east-1`** u **Ohio `us-east-2`**).
